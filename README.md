# WowDB_zhCN
魔兽世界 3.3.5 的数据库汉化sql，支持 TrinityCore 的数据库翻译。
汉化内容来自网络，我只是对sql脚本做了 TrinityCore 的适配。
因为来源不可靠，所以汉化内容不太“纯净”，大部分sql是台湾繁体版本的翻译，但是也混杂了几个国服简体的sql。
暂时没找到更可靠的汉化文件，所以目前也只能这样了。
